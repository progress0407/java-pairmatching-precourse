# 구현할 기능 목록 정리

---

## 비즈니스 로직

- 페어는 두명씩 매칭
- 홀수인 경우 한 페어는 3인 구성
  - 예) `리사 : 덴버 : 제키`
- 같은 레벨에서 이미 페어를 맺었다면 이 다음에는 페어가 될 수 없다
- 

- [ ] md 파일을 읽는 파일 읽기 처리 클래스

## 입력부
- [ ] 최초 진입 입력부를 구현
- [ ] 두번째 진입 입력부 구현

## 예외처리
- 사용자가 잘못된 입력을 할 경우
  - [ ] `IllegalArgumentException` 예외
  - [ ] `[ERROR] `로 시작하는 메세지
  - [ ] 해당 입력부터 다시 입력
